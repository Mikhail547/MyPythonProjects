# 💻 Моё портфолио решений: ЕГЭ и Codeforces

Добро пожаловать в репозиторий с моими решениями алгоритмических задач! Здесь собраны материалы моей подготовки к экзаменам и соревнованиям по спортивному программированию.

## 📊 Что внутри репозитория:
* **Задачи ЕГЭ по информатике** — решения актуальных номеров (включая сложные задания 24, 25, 26, 27(такие задачи как 26, 27 я решал в LibreOffice Calc в дальнейшем добавлю в репозиторий)) с разбором файлов данных.
* **Codeforces контесты** — скрипты для олимпиадных задач разного уровня сложности.
* **Базовые алгоритмы** — работа со стандартным вводом `sys.stdin`, кастомная сортировка, обходы графов и динамическое программирование.

## 🛠️ Стек технологий:
* **Языки и библиотеки:** Python 3.13 (включая `turtle` для геометрических задач, `sys`, `math`)
* **Инструменты автоматизации:** MS Excel / LibreOffice Calc (динамическое программирование, фильтры и формулы)
* **Среда разработки:** PyCharm Professional
